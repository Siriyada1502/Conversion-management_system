# ระบบจัดการแปลง React + Leaflet

## วิธีรันโปรเจกต์

```bash
npm install
npm run dev
```

จากนั้นเปิดลิงก์ที่ Vite แสดง เช่น

```bash
http://localhost:5173
```

## ความสามารถของระบบ

- อ่านข้อมูลจาก `src/data/plot.json`
- แสดงแปลงทั้งหมดบนแผนที่ Leaflet
- แปลงพิกัด UTM Zone 47N เป็น Lat/Lng เพื่อแสดงบนแผนที่
- คำนวณพื้นที่ด้วย Shoelace Formula
- แสดงจุดกึ่งกลางของแต่ละแปลง
- ค้นหาแปลงด้วยเลข `plot_id`
- เพิ่มแปลงใหม่โดยคลิกบนแผนที่อย่างน้อย 4 จุด
- บันทึกข้อมูลไว้ใน Local Storage
- ดาวน์โหลด JSON ที่อัปเดตเป็น `plot-updated.json`

## หมายเหตุ

เว็บ Frontend อย่างเดียวไม่สามารถเขียนทับไฟล์ `src/data/plot.json` ในเครื่องได้โดยตรง เพราะ Browser ถูกจำกัดความปลอดภัย  
จึงใช้วิธีบันทึกใน Local Storage และให้กดดาวน์โหลด JSON ที่อัปเดตแทน
